# Chess Plus - Coursework

Веб-платформа для гри в шахи в реальному часі з системою рейтингу та профілями користувачів.
Розроблено в рамках курсової роботи.

## Технологічний стек

* **Frontend:** Vue.js 3, Vite, CSS3.
* **Backend:** Node.js, Express.
* **Sync:** Socket.io.
* **Database:** MongoDB.
* **Logic:** Chess.js (валідація ходів).

## Функціонал (v1.1)

* **Мультиплеєр:** Гра онлайн через WebSockets.
* **Авторизація:** Реєстрація та вхід, збереження сесії.
* **Профілі:** Кастомізація аватара, біо, статистика.
* **Рейтинг:** Система ELO.
* **Стабільність:** Захист від розриву з'єднання, перепідключення до гри.
* **Інтерфейс:** Адаптивний дизайн, налаштування тем та графіки.

### Попередні вимоги запуску
* Встановлений **Node.js**.
* Запущена служба **MongoDB** (локально на порту 27017).
